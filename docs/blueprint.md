# **App Name**: Eternal Flame

## Core Features:

- Photo Gallery: Display a collection of photos with captions.
- Shared Memories Timeline: Display a chronological timeline of shared memories.
- Love Letter Generator: Use a generative AI tool to draft heartfelt messages. It considers previous messages and memories for personalization.
- Special Dates Reminder: A reminder for anniversaries and birthdays.
- Favorite Things List: List of her favorite songs, movies, places, and things.

## Style Guidelines:

- Primary color: Soft Lavender (#D1B0FF) to evoke tenderness and romance.
- Background color: Very light purple (#F8F5FF). Close in hue to the primary, but almost fully desaturated for a gentle feel.
- Accent color: Dusty Rose (#DFA2AB), an analogous color, providing a gentle contrast that emphasizes romance.
- Headline font: 'Playfair', a modern serif similar to Didot, lending a high-end and elegant feel.
- Body font: 'PT Sans', a humanist sans-serif used to pair with 'Playfair', is both modern and warm for longer passages of text.
- Use elegant and minimalistic line icons, related to love and relationships.
- Soft and subtle transitions for a romantic user experience.
